# shiro-redisson
```
使用redisson包解决redis缓存，解决shiro-redis使用*查询时，导致redis长时间卡死
```
## JDK
> 1.8

# 如何使用？
1. 引入Maven依赖或下载jar包

``` xml
        <dependency>
            <groupId>com.github.fartherp</groupId>
            <artifactId>shiro-redisson</artifactId>
            <version>1.0.0</version>
        </dependency>
```
